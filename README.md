# Menu App

## About

Menu app is a beginner Flutter app created to get familiar with some common widgets like 
Cards, CircleAvatar, Drawer, FloatingActionButton, ThemeData, ListView.


### Widgets
* ThemeData
* BottomNavigationBar
* SwitchListTile
* SingleChildScrollView
* GridView
* ListView
* ListTile
* Cards
* Drawer
* FloatingActionButton
* CircleAvatar
* FittedBox
